【   ソフト紹介   】
      本ソフトは家計簿ソフトです。
      家計簿ソフト 記帳風月 から多くのデータの移行が可能です。
      データベースとして、SQLiteおよびPostgreSQLに対応しています。また、それらの
    相互乗り換えも可能です。
      DBアクセスを高速に実行したいのであればPostgreSQLを、手軽に利用したい場合は
    SQLiteをお勧めします。

【  インストール  】
    圧縮ファイルを解凍

【アンインストール】
    解凍されたファイルを全て消去

【     使い方     】
    以下を参照
      https://github.com/vtr00/HouseholdAccountBook/blob/master/README.md
